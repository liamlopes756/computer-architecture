# MIPS Assembly Study Projects

This repository contains a collection of assembly language programs developed for studying computer architecture, MIPS assembly, and memory management concepts.

The projects focus on understanding how low-level instructions work, how data is stored and accessed in memory, and how the MIPS architecture handles registers, control flow, stack usage, and memory allocation.

## Purpose

This repository was created for academic and practical study of:

- Computer architecture fundamentals
- MIPS assembly language programming
- Memory organization and allocation
- Register usage and instruction execution
- Low-level problem solving

## Concepts Covered

The programs in this repository are designed to explore topics such as:

- Conditional branches and loops
- Function calls and stack usage
- Input and output routines
- Static and dynamic memory concepts
- Data manipulation at the assembly level
- Arithmetic and logic operations


## Technologies

- Assembly Language
- MIPS Architecture
- MARS Simulator


## How to Run

1. Download and open **MARS Simulator** https://dpetersanderson.github.io/
2. Load any `.asm` file
3. Click **Assemble**
4. Run the program or execute step-by-step
